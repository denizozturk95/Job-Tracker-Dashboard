# Job-Tracker-Dashboard
Overview
I built this to keep track of my job applications without relying on messy spreadsheets or scattered notes. When you're applying to a lot of roles, it gets hard to remember where you applied, what stage you're at, and what needs follow-up—this helps keep everything in one place.

### What It Does
Lets me log job applications with basic info like company, role, and date
Shows all applications grouped by status (applied, interviewing, etc.)
Makes it easy to update where I am in the process
Has search/filtering so I can quickly find specific applications
Allows adding notes for things like interview prep or contacts
Keeps track of important dates like interviews or follow-ups
### Why I Built It

I wanted something simple to manage my job search properly. I kept losing track of applications and missing small details, so I made a tool that gives me a clearer overview and keeps things organized.
